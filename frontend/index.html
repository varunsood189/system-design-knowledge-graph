System Design Knowledge Graph

Turn engineering blogs into a connected learning graph — you do not need to read the full article.

Why the blog URL? The blog is the source document. We download it, extract clean text, then run 3 LLM steps: (1) concepts with quotes from the text, (2) how concepts relate, (3) what to learn next. Results merge into data/graph.json — a persistent graph that grows with every article you ingest. You can learn from the evidence snippets below without opening the blog; the link is there if you want depth.
